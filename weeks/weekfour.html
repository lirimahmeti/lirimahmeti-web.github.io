Selectors

Types of selectors

  • UNIVERSAL SELECTOR
    • Example: *{}
  • TYPE SELECTOR
    • Example: h1, h2, h3, p, span...
  • CLASS SELECTOR
    • Example: .classvalue{}, p.classvalue{}
  • ID SELECTOR
    • Example: #idvalue{}
  • CHILD SELECTOR
    • Example: li > a{}
  • DESCENDANT SELECTOR
    • Example: p a{}
  • ADJACENT SIBLING SELECTOR
    • Example: h1 + p{}
  • GENERAL SIBLING SELECTOR
    • Example: h1 ~ p{}

    Attribute selectors

    Types of attribute selectors

  • "Begins with" SELECTOR
    • Example: [attr^="val"]
  • "Ends with" SELECTOR
    • Example: [attr$="val"]
  • "Contains" SELECTOR
    • Example: [attr*="val"]
  • "Equals" SELECTOR
    • Example: [attr="val"]
  • "Attribute" SELECTOR
    • Example: [attr]
  • SPACE SELECTOR
    • Example: [attr~="val"]
