What are pseudo classes used for?

A pseudo-class is used to define a special state of an element. For example, it can be used to: Style an element when a user mouses over it. Style visited and unvisited links differently.

What are pseudo elements used for?

A CSS pseudo-element is used to style specified parts of an element. For example, it can be used to: Style the first letter, or line, of an element. Insert content before, or after, the content of an element.

How to define size of elements

We can define size of elements using the width/height propperties. Keep in mind that we cant use the height/width property on inline elements

Ways of displaying elements

display:inline tells the element to fit itself on the same line.

display:block Is an element that starts on a new line and takes up the available screen width. You can specify the width and height properties for such elements.

display:inline-block Compared to display: block , the major difference is that display: inline-block does not add a line-break after the element, so the element can sit next to other elements.

Ways of displaying elements

position: relative changes the position relatively to the object parent

position: absolute changes the position relatively to the viewport

position: fixed same as absolute except that it doesnt change position when scrolling

position: sticky its position remains unchanged while scrolling until the object touches the defined top of the viewport

Visibility of elements

visibility: visible changes the position relatively to the object parent

visibility: hidden changes the position relatively to the viewport

visibility: collapse same as absolute except that it doesnt change position when scrolling

visibility: initial its position remains unchanged while scrolling until the object touches the defined top of the viewport

visibility: inherit its position remains unchanged while scrolling until the object touches the defined top of the viewport

Defining CSS variables

Variables in css have the same logic as in every other programming language. They are pretty good tools to use since we can centralize our code and then when we need to make changes we can easily make them from one place.

Variables in CSS3 are defined in the :root and to define a variable we just write before the variable name this --variable-name: value;

Importing

We learned about the import and link function. While there are alternatives, CSS @import is the quickest and most accurate way to bring already existing styles from one style sheet to another.

Another alternative are: Links : Using a link tag to link your CSS files within the header of your HTML allows CSS files to be downloaded rather than called.
